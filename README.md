# 🌲 Sunny Land Forest

**Sunny Land Forest** é um jogo de aventura 2D em estilo **pixel art**, desenvolvido utilizando a **Unity**.

## 🎮 Sobre o Projeto

Sunny Land Forest é um jogo focado em exploração e aventura, onde o jogador percorre cenários em um ambiente estilizado, enfrentando desafios e interagindo com o mundo ao seu redor.

O projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento de jogos, incluindo mecânicas, física, animação e design de fases.

## 🛠️ Tecnologias Utilizadas

- Unity (Game Engine)
- C#
- Pixel Art

## ⚙️ Funcionalidades

- Movimentação do personagem
- Sistema básico de física
- Animações 2D
- Interação com o ambiente
- Estrutura de fases

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, trabalhei com:

- Criação de mecânicas de gameplay
- Sistema de movimentação em 2D
- Animações na Unity
- Organização de cenas e objetos
- Lógica de jogo utilizando C#
